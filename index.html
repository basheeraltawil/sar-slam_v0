The Problem

Moving objects break traditional SLAM systems. Their features violate geometric assumptions, causing:

  • ‚ùå Wrong motion estimates
  • ‚ùå Accumulated drift
  • ‚ùå Tracking failure in crowded areas
  • ‚ùå Loss of useful static features

Our Solution

üí° Smart Motion Detection

SAR-SLAM doesn't just detect objects‚Äîit checks if they're actually moving. A standing person? Keep their features. A walking person? Filter only their moving features.

How it works: ORB feature tracking + RANSAC homography detects outliers. If >30% of an object's features are outliers, it's moving‚Äîfilter those features only.

‚ö° Adaptive Processing

  • Empty scenes: Semantic detection only
  • Few people: Geometric verification only
  • Crowded: Full semantic + geometric fusion
